# Open Source Badge Dashboard

> Github Dashboard with badge using [badgen.net](https://badgen.net/).

## [Demo](https://emasuriano.github.io/oss-badge-dashboard)

## Setup

- Clone the repository into your account
- Open the [config file](./config.json) to customize your projects and the columns you want to show.
- Commit your changes into the repository.
- Github will execute `yarn build` and auto-commit the `index.md` file.
